- парсинг по используемым технологиям можно разделить на **два вида**:  
  - статические страницы  
  - динамические страницы  

- курсовик посвящён **статическому парсингу**  
- для этого сначала нужно загрузить html-страницу  
- разобраться в тегах и найти где именно находится полезная информация  
- далее каким-то доступным "достать" эти данные  
  - с помощью срезов в строках  
  - с помощью регулярных выражений  
  - с помощью специализированных библиотек  
- далее из полученных данных сформировать коллекции объектов  
- затем сохранить в требуемый формат (json, csv, БД, ...)
