### Задание labrab-05  

- переложить все или часть методов (смотрите сами, по возможности),  
- которые вы делали для классов Библиотеки, Пользователя и Книги
- на приложение написаннное на серверном фреймворке `Express.js`  
- в прошлой Лабе данные и команды передавали через строку `Терминала`  
- в этой лабе данные передавать через `адресную строку Браузера`  
- или через `ThunderClient`, `Postman`  
- для `POST` запросов только через `ThunderClient`, `Postman`  
- в Express можно использовать объекты:  
  - `req.params`  
  - `req.query`  
  - `req.body`
- `результаты выводить` в окно Браузера  
- можно `для контроля` выводить в консоль  

---  

Готовое приложение запускаете в терминале или в среде разработке.  
Получение данных (`GET`) через адресную строку Браузера или через `ThunderClient`, `Postman`.  
Добавление (`POST`), изменение (`PUT`) или удаление (`DELETE`) данных через `ThunderClient`, `Postman`.
