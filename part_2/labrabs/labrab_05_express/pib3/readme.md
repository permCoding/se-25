#### 03

- нужно создать метод POST  
- будем добавлять в массив новый объект  
- у существующих объектов есть поля `id` и `title`  
- в исходном файле объекты неотсортированы  
- но их id-шники создавались последовательно - от 1 до 11  
- просто потом файл был перемешан  
- ваш метод POST по пути `/users/addUser`  
- должен принимать объект `{ "title": "тут_название" }`  
- после чего искать в массиве существующих объектов объект с максимальным id  
- и добавлять в массив новый объект с номером id на единицу больше  

например - был такой массив объектов:  

```json
[
    { "id": 3, "title": "d9sT" },
    { "id": 4, "title": "xZvN2" },
    { "id": 6, "title": "hR7jK" },
    { "id": 7, "title": "tGpQw" },
    { "id": 8, "title": "student" },
    { "id": 9, "title": "bVrS5" },
    { "id": 10, "title": "lMzP9" },
    { "id": 5, "title": "aBcDeF" },
    { "id": 11, "title": "gHijK" },
    { "id": 1, "title": "qW8k" },
    { "id": 2, "title": "mYpLr" }
]
```

пользователь добавляет такой объект `{ "title": "JavaScript" }`  

но в массиве добавляется такой:  

```json
  {  
    "id": 12,
    "title": "JavaScript"
  }
```
