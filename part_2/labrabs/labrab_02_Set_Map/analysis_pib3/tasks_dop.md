### task_dop_01  

В сторке находится случайный набор цифр.  

Требуется все цифры считать из строки, отсортировать так:  

- чтобы вначале строки шли все чётные цифры по возрастанию  
- а далее все нечётные по убыванию  

Пример входной строки:  

```js
const str = '01229722';
```

Для которой правильный ответ `02222971`.  

Программу писать в функциональном стиле с отсутпами.
