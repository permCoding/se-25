- ***массив; множество: Set(); словарь: объект, Map()***  
- `обработка файлов`  

---  

Дан `.csv` файл (количество строк не более 10000).  
В каждой строке файла шесть натуральных чисел (через пробел).  
Нумерация строк в файле начинается с единицы.  

### Задание 01  

Требуется вывести на экран `номера строк`,  
для чисел которых выполнены все перечисленные условия:  

1) все числа в строке нечётны и различны;  
2) и расположены в порядке возрастания.  

> рекомендуется использовать `Set()`, `.toSorted()`  

---  

### Задание 02  

Требуется вывести на экран `номера строк` и их `содержимое` (для контроля),  
для чисел которых выполнены все перечисленные условия:  

1) в строке есть одно число, которое повторяется трижды, а остальные числа различны;  
2) повторяющееся число строки больше, чем среднее арифметическое её неповторяющихся чисел.  

> рекомендуется использовать словарь в виде `обычного объекта как совокупности пар <ключ:значение>)`  

---  

### Задание 03  

Требуется вывести на экран `номера строк` и их `содержимое` (для контроля),  
для чисел которых выполнены все перечисленные условия:  

1) в строке есть ДВА числа, которые повторяются ДВАЖДЫ, а остальные числа различны;  
2) сумма повторяющихся чисел строки МЕНЬШЕ, чем сумма неповторяющихся чисел.  

Примеры:  

- `22 34 22 31 10 10` - подходит по обоим условиям;  
- `22 31 22 31 80 80` - не подходит по первому условию;  
- `22 16 22 31 80 80` - не подходит по второму условию.  

> рекомендуется использовать словарь в виде `объекта типа Map()`
