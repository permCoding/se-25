## Data structures  

В JS (Node.js) кроме примитивов (string, number, bigint, boolean, null, undefined, symbol) используются объекты - ссылочные типы.  
Примитивы передаются `по значению`, объекты `по ссылке`.  
Примитивы - неизменяемы (immutable), объекты - изменяемы (mutable).

### Объект Set  

#### Основные методы Set  

- `new Set([iterable])` - конструктор создаёт множество из итерируемого объекта  
- `.add(value)` - добавляет значение (дубликаты игнорируются)  
- `.has(value)` - проверяет наличие значения, возвращает `true/false`  
- `.delete(value)` - удаляет значение, возвращает true при успехе  
- `.clear()` - очищает множество  
- `.size` - количество элементов  
  
#### Итерационные методы Set  

- `.keys()` - возвращает итератор ключей  
- `.values()` - возвращает итератор значений  
- `.entries()` - возвращает итератор пар [value, value]  
- `.forEach(callback[, thisArg])` - перебирает элементы с колбэком  

#### Методы работы со множествами  

- `.intersection(other)` - возвращает пересечение с другим Set  
- `.union(other)` - возвращает объединение  
- `.difference(other)` - возвращает разность  
- `.symmetricDifference(other)` - симметрическая разность  
- `.isSubsetOf(other)` - проверяет, подмножество ли это  
- `.isSupersetOf(other)` - проверяет, supermultiset ли  
- `.isDisjointFrom(other)` - проверяет отсутствие общих элементов

### Объект Map  

- это коллекция пар "ключ-значение" с любыми типами ключей
- порядок вставки - сохраняется  

#### Основные методы Map  

- `.new Map([iterable])` - конструктор создаёт Map из итерируемого объекта пар  
- `.set(key, value)` - добавляет/обновляет пару ключ-значение  
- `.get(key)` - возвращает значение по ключу или undefined  
- `.has(key)` - проверяет наличие ключа, возвращает `true/false`  
- `.delete(key)` - удаляет пару по ключу, возвращает `true` при успехе  
- `.clear()` - очищает всю Map  
- `.size` - количество элементов  

#### Итерационные методы Map  

- `.keys()` - итератор всех ключей  
- `.values()` - итератор всех значений  
- `.entries()` - итератор пар [key, value]  
- `.forEach(callback)` - перебор с колбэком  
- `Map()` - поддерживает for...of напрямую
