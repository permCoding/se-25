## задание 0  

```js
[ 
  '888_123_345',
  '888', // true
  '1234_345',
  '12_234',
  '1_234',
  '12_23',
  '12',
  '9',
  '12_3',
  '12_',
  '035_709', // false
]
```

для каждого элемента массива проверить с помощью регулярки  
используя метод re.text(str) является ли элемент корректным 
числом формата с подчёркиванием, например,  
`12_549`, `888`, `9`, `888_777_009` - корректно (true)  
`03_345`, `1234_345`, `123_34` - НЕкорректно (false)

## задание 1  

```js
log('id000_345_888 12_567 123 043_909 888_999_132'.replace(/_/g, ''));
// 'id000_345_888 12567 123 043_909 888999132' // правильный ответ
```

- в строке есть числа, заданые в формате с подчёркиваниями 888_123_345  
- в строке есть и другие сочетания цифр с символами  
- заменить корректно написанные числа с подчёркиваниями  
- на обычное представление (числа без подчёркиваний)  
- например: `12_345`, `9` - корректное представление  
- `034_456`, `2345_678`, `id12_234` - НЕкорректные

## задание 2  

```txt
дан массив строк, в которых
задана температура в градусах Цельсия или Фаренгейта  
требуется распарсить данные и 
сделать перевод в градусы Фаренгейта (или Цельсия)  
- градусы Фаренгейта могут обозначаться буквами F или f  
- градусы Цельсия могут обозначаться буквами C или c  
- градусы могут быть целым или вещественным числом  
```

примеры входа:  

```js
let arr = [
    '+11.8F',
    '+11.8f',
    '10C',
    '10    C',
    ' - 11.8   C '
];
```

формулы перевода:  
`F = С * 9 / 5 + 32`  
`C = (F - 32) * 5 / 9`  

```txt
на вход подайте приведённый выше массив строк  
пусть программа с помощью регулярки распарсит 
числовое значение температуры и вид измерения (F, C)
далее можно использовать обычный if () { }
и вычисление по приведённым формулам
```

## задание 3  

Пусть дана строка:  

```js
const str = "Припекало жаркое Алжирское солнце, температура в тени держалась на уровне +95F, кирпичные поверхности домов раскалялись до +114.5F. Мне нездоровилось - градусник показывал 98.96F. Сравните с температурой открытого космоса, равной -459.76F.";
```

Требуется с помощью регулярок и метода `.replace()` заменить ВСЕ представления градусов в Фаренгейтах, на представления градусов по шкале Цельсия с добавлением символа градуса °.  Пример корректной записи в итоговой строке `... -273.2°С ...`.  

Вывести на экран обновлённую строку для проверки.
