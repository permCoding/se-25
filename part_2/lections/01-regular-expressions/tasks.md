## задание 1  

В массиве даны целы числа в диапазоне от 1 до 999_999.  
В массиве есть некорректно заданные значения - они не соответствуют способу задания целых чисел с разрядными группами по три цифры.  
Правильно заданное число не должно начинаться с нуля и не может содержать в левой группе цифр более трёх цифр или менее одной (в массиве такие числа обозначены false).  
В массиве нет чисел, содержащих более двух групп цифр, например недопустимое число - 12_345_789.  

```js
[ 
  '888_123',
  '888', // true
  '1234_345',  // false - не может быть более 3-х цифр в первой группе
  '12_234',
  '1_234',
  '12_23',  // false
  '12',
  '9',
  '12_3',  // false
  '035_709',  // false - не может начинаться с 0
  '23_',  // false
  '_23',  // false
]
```

Для каждого элемента массива (используя .forEach()) требуется проверить  
с помощью регулярки (используя метод re.test(str))  
является ли элемент корректным числом формата с подчёркиванием, например,  
`12_549`, `888`, `9`, `888_777` - корректно (true)  
`03_345`, `1234_345`, `123_34`, `12_345_909` - НЕкорректно (false)

## задание 2  

Пусть дана строка, содержащая числа, оформленные в стиле с разделением на группы по три цифры по линиям подчёркивания, например, `12_200`. В строке есть другие сочетания цифр с символами и просто текст. Требуется заменить только корректно написанные числа с подчёркиваниями на обычное представление (числа без подчёркиваний). В тексте могут находиться большие целые числа (больше чем две цифровые группы) - от 1 до 10**30, то есть 123_123_666_999_000_023 - допустимое число.  
Правильно заданное число не должно начинаться с нуля и не может содержать в левой группе цифр более трёх цифр или менее одной (в массиве такие числа обозначены false).  
Например: `12_345`, `9`, `888_999_132` - это корректно представленные числа (их формат нужно поменять).  
Но `034_456`, `2345_678`, `id12_234`, `_203`, `203_` - НЕкорректные (их не надо заменять).  
Предполагается, что некорректно заданные числа не являются числами, а несут другую смысловую нагрузку в тексте (идентификационные номера, другие обозначения).  

```js
const str = 'пример входной строки: id000_345_888 12_567 123 043_909 888_999_132 _23 23_';
// 'пример входной строки: id000_345_888 12567 123 043_909 888999132 _23 23_' // правильный ответ
```

## задание 3  

```txt
дан массив строк, в которых
задана температура в градусах Цельсия или Фаренгейта  
требуется распарсить данные и 
сделать перевод в градусы Фаренгейта (или Цельсия)

- градусы Цельсия могут обозначаться буквами C или c  
- градусы могут быть целым или вещественным числом  
```

примеры входа:  

```js
let arr = [
    '+11.8F',
    '+11.8f',
    '10C',
    '10    C',
    ' - 11.8   C '
];
```

формулы перевода:  
`F = С * 9 / 5 + 32`  
`C = (F - 32) * 5 / 9`  

```txt
на вход подайте приведённый выше массив строк  
пусть программа с помощью регулярки распарсит 
числовое значение температуры и вид измерения (F, C)
далее можно использовать обычный if () { }
и вычисление по приведённым формулам
```

```js
const str = "Припекало жаркое Алжирское солнце, температура в тени держалась на уровне +95F, кирпичные поверхности домов раскалялись до +114.5F. Мне нездоровилось - градусник показывал 98.96F. Сравните с температурой открытого космоса, равной -459.76F.";
```

Вывести на экран обновлённую строку для проверки.
